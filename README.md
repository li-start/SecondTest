# SecondTest
开发环境
=======
 - 开发工具：pycharm
 - 编程语言：python3
 - 使用方法：A*算法
 
 运行
======
  在main.py文件中运行可以自动申请题目和提交答案
  在AI_test.py文件中，修改json申请部分代码中的url的uuid值，运行后，可以做AI大比拼
  在play.py文件中运行可以得到原型设计实现的小游戏
